# Marketplace Platform for Fishermen

Agile/Scrum-based marketplace platform connecting fishermen with restaurants and seafood buyers.

## Features
- Seafood listing & inventory management
- Ordering & payment workflows
- User authentication
- REST API integration
- Automated testing with Cypress

## Tech Stack
- Frontend: HTML/CSS/JavaScript
- Backend: Java
- Database: SQL
- Testing: Cypress
- DevOps: Docker, Docker Compose

## Responsibilities
- Backend workflow development
- SQL schema design
- Frontend-backend integration
- Unit/system/automated testing

## Development Process
- Agile/Scrum
- Sprint planning
- Daily standups
- Iterative delivery
